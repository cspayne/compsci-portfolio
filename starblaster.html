StarBlaster Microcontroller

Cypress Payne

  • Language: C
  • Skills: Microcontrollers, Embedded Systems, Combined Hardware and Software Solutions
  • View Repository

This project was the beginning of a game implementation using PSoC Creator to interface with a PSoC Pioneer Kit and a GLCD screen. I produced multiple versions of the game interface, with the crosshairs being controlled by ABCD buttons from a joystick board in the first version, and later, by implementing tilt control using an accelerometer.
