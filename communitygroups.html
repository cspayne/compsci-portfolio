Community Groups Algorithm

Cypress Payne and Zachary Himes

  • Language: Python
  • Skills: Algorithm Design and Analysis, Object-Oriented Programming, Greedy Algorithms
  • View Repository

Our task was to create an algorithm when given a group of people, was able to sort that group into equal groups of a size given by the user. The restrictions for the groups were as follows: Must be as evenly distributed as possible, there should be a host for each group, married couples must always remain in the same group, and everyone should visit everyone else's house (Everyone needs to host everyone else at least once). We utilized Python to implement this algorithm, as it had a nice array of graph libraries we would be able to use to assist us in both the implementation and display of the algorithm.
